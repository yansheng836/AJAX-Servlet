AJAX-jQuery的常用方法总述

1、$().load(url, [data], [callback])

通过 HTTP 请求加载远程数据。
jQuery 底层 AJAX 实现。简单易用的高层实现见 $.get, $.post 等。$.ajax() 返回其创建的 XMLHttpRequest对象。大多数情况下你无需直接操作该函数，除非你需要操作不常用的选项，以获得更多的灵活性。 最简单的情况下，$.ajax()可以不带任何参数直接使用。

常用参数简介： 
load(url, [data], [callback])

data 参数规定与请求一同发送的查询字符串键/值对集合。
callback 参数规定当 load() 方法完成后所要允许的回调函数。回调函数可以设置不同的参数：

注意：数据传递方法

1.简单、不传递数据的load方法：
    load("../SimpleServlet", function (responseTxt, statusTxt, xhr) {})

2.传递数据的load方法：
     2.1 通过url传递数据，在1的基础上拼接url。
            如：var url =  "../PostServlet?"+"fname=AlanMathison&lname=Turing";

    2.2 用键值对传递数据。
             如：load("../PostServlet",{key:value,key1:value1} function (responseTxt, statusTxt, xhr) {})

2、jQuery.ajax(url,[settings])

通过 HTTP 请求加载远程数据。
jQuery 底层 AJAX 实现。简单易用的高层实现见 $.get, $.post 等。$.ajax() 返回其创建的XMLHttpRequest对象。 大多数情况下你无需直接操作该函数，除非你需要操作不常用的选项，以获得更多的灵活性。最简单的情况下，$.ajax()可以不带任何参数直接使用。

常用参数简介： 
$.ajax({
    url: url,
    async: true,      //是否同步，默认为true
    type: "GET",      //请求方式 (常用"POST" 或 "GET")，默认: "GET"
    data: Object,String //发送到服务器的数据。将自动转换为请求字符串格式。GET 请求中将附加在 URL 后。查看 processData 选项说明以禁止此自动转换。
                        //必须为 Key/Value 格式。如果为数组，jQuery 将自动为不同值对应同一个名称。
    dataType: "text", //预期服务器返回的数据类型。如果不指定，jQuery 将自动根据 HTTP 包 MIME 信息来智能判断，比如XML MIME类型就被识别为XML。
                      // 可选参数："xml","html","script","json","jsonp","text"
    cache: false,     //(默认: true,dataType为script和jsonp时默认为false) jQuery 1.2 新功能，设置为 false 将不缓存此页面。
    success: function (responseTxt, statusTxt, xhr) { //success(data, textStatus, jqXHR),请求成功后的回调函数。
        ……      //responseTxt - 包含调用成功时的结果内容,statusTXT - 包含调用的状态,xhr - 包含 XMLHttpRequest 对象
    },
    error: function (responseTxt, statusTxt, xhr) {  //error(data, textStatus, jqXHR),请求失败后的回调函数。
        ……
    }
});

3、jQuery.get(url, [data], [callback], [type])

通过 HTTP 请求加载远程数据。
jQuery 底层 AJAX 实现。简单易用的高层实现见 $.get, $.post 等。$.ajax() 返回其创建的 XMLHttpRequest对象。大多数情况下你无需直接操作该函数，除非你需要操作不常用的选项，以获得更多的灵活性。 最简单的情况下，$.ajax()可以不带任何参数直接使用。

常用参数简介： 
jQuery.get(url, [data], [callback], [type])
url:待载入页面的URL地址
data:待发送 Key/value 参数。
callback:载入成功时回调函数。
type:返回内容格式，xml, html, script, json, text, _default。
